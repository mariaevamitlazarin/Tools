# 🍏 Eva's Tools

> 💡 _A lighthouse for developers, scientists & researchers — small tools that make the everyday a little brighter._

💖 **Welcome!** This is my main public repository: a growing collection of examples and functions to help with your day-to-day work as a **developer**, **scientist**, or **researcher**.

> [!note]
> Some sources you need to install on your PC ou Mac. By the way... Everything was tested on a Mac only. For Windows or other operating systems, you'll just need some adaptations.

> [!tip]
> `Tools` is the **main branch**. Everything here is shared openly — built to be reused, improved, and valued by the community. Each side branch focuses on a specific kind of work, so you can pick exactly what you need. > 🤝 **My commitment:** to share freely, to keep things useful, and to value every contribution and idea that helps this repo grow.
